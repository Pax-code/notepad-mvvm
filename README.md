
# NotePad MVVM
This project incorporates various technologies to achieve its functionality. The key technologies used are: 

<br/>

##### -Room
##### -Navigation
##### -LiveData
##### -RecyclerView
##### -ViewBinding 
##### -Fragments
##### -Customized Alert Dialogs
